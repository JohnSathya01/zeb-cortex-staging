# zeb-cortex-staging

> Cortex — Learning · Capability · Excellence  
> Powered by **zeb labs**

A modern LMS (Learning Management System) built with React + Firebase, designed for managing learners, courses, assessments, and peer reviews.

## Tech Stack

- **Frontend:** React 18, Vite
- **Backend/DB:** Firebase Realtime Database
- **Auth:** Firebase Authentication
- **Hosting:** Firebase Hosting
- **CI/CD:** GitHub Actions

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy

Pushes to `main` are automatically built and deployed to Firebase Hosting via GitHub Actions.

> Set `FIREBASE_SERVICE_ACCOUNT` in GitHub repository secrets to enable automated deploys.
